# Internship---Data-Science-AIML-
A collection of machine learning projects completed during my internship, featuring a Movie Recommendation System (Content-based) and Sentiment Analysis using NLP.

**1. Movie Recommendation System**

  Goal: Suggest movies to users based on viewing history.
  Tech Stack: Python, Pandas, Scikit-learn, Cosine Similarity.
  Key Result: Recommended 6 Movies similar in taste for each Movie.

**2. Sentiment Analysis Model**

  Goal: Classify text data as positive, negative, or neutral.
  Tech Stack: Python, NLTK, Scikit-learn, Classification Report.
  Key Result: Successfully processed 2 reviews with 89% accuracy.
